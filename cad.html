Page for Notes on AutoCAD and Carlson Products

This is just a list of notes on AutoCadd and Carlson Products I use.

AutoCAD

  • AutoCAD Map 3D 2024 Getting Started - Good Place To Get Started on AutoCAD Map 3D

Carlson

  • Carlson Software's Training Webinars - Recorded Webinars on Carlson Software
  • Getting Started With Carlson - Best Place To Start when getting familiar with Carlson software. Is an introductory webinar, recorded in 2022, that covers Carlson specific settings for both projects and individual files, importing points and various tools to display/edit points from a crd file. Really good training to get you feed wet with the Carlson software package.
  • Creating Carlson Surfaces - Best place to start when getting familiar creating surfaces in Carlson software. Is an introductory webinar, recorded in 2022, that covers Carlson specific functions for creating surfaces from collected survey points and also gives some idea on how that can be incorporated with field to finish.
  • Existing and Design Modeling Made Easy With SiteNET - Introductory use of Carlson SiteNET tools. Shows the Carlson tools that you can use to design existing surfaces as well as proposed surfaces. The examples given is a building pad, parking lot, and access road placed on existing ground.
  • Merging Tins - Use of Carlson tools for merging tin files.
